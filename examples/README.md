# LeanSpec Examples

For now, check out:
- `../templates/` - Project initialization templates
  - `solo-dev/` - Quick setup for individual developers
  - `team/` - Small team collaboration
  - `enterprise/` - Enterprise-grade with governance
  - `api-first/` - API-driven development
- `../src/` - TypeScript CLI implementation
- `../AGENTS.md` - How AI agents should work with this project
- `../specs/20251031/` - Actual specs used in this project

Initialize LeanSpec in your project:

```bash
# Interactive setup
lspec init

# List available templates
lspec templates

# Create your first spec (creates folder structure)
lspec create my-feature
```
