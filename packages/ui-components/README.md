# @leanspec/ui-components (Deprecated)

This package is deprecated. Use `@leanspec/ui` instead.

## Migration

```tsx
import { StatusBadge } from '@leanspec/ui';
import '@leanspec/ui/styles.css';
```

If you previously imported styles from `@leanspec/ui-components/styles.css`, switch to `@leanspec/ui/styles.css`.
